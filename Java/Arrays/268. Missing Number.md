# Approach

## Brute Force

- Create an Array (arr) of size n+1.
- Fill it with -1
- Iterate through original array and replace the value in arr with value found in original array with index as value of original array.
- Iterate through arr, if any value found is -1, return it's index.
- If entire arr is filled without -1, return -1.

```java
class Solution {
    public int missingNumber(int[] nums) {
        int[] arr=new int[nums.length+1];
        Arrays.fill(arr,-1);
        for(int i=0;i<nums.length;i++){
            arr[nums[i]]=nums[i];
        }
        for(int i=0;i<arr.length;i++){
            if(arr[i]==-1) return i;
        }
        return -1;
    }
}
Time Complexity: O(n)
```

## Boolean Array Approach

- Create a Boolean Array of size n+1.
- Iterate through the given elements and mark true in the Boolean Array if seen.
- Iterate through the Boolean Array, if found false return that index.
- If entire array is true, return -1.

```java
class Solution {
    public int missingNumber(int[] nums) {
        boolean[] seen=new boolean[nums.length+1];
        for(int num:nums){
            seen[num]=true;
        }
        for(int i=0;i<seen.length;i++){
            if(!seen[i]) return i;
        }
     return -1;
     }
}
Time complexity: O(n)
```

## Sum of Natural numbers

- Subtracting sum of natural numbers (n=length of the Array) - sum of given elements in the array results the Missing number.

```java
class Solution {
    public int missingNumber(int[] nums) {
        int res=0,n=nums.length;
        int sum=(n*(n+1))/2;
        for(int num:nums){
            res+=num;
        }
        return sum-res;
    }
}
Time Complexity: O(n)
```

## XOR Approach - Bit Manipulation

- A number XORed with itself returns 0, X^X=0
- A number XORed with 0 returns X, X^0=X
- Calculate the XOR of given range and the XOR of elements in the array
- Calculate XOR from the results - Missing Number.

```java
class Solution {
    public int missingNumber(int[] nums) {
        int rangexor=0;
        int arrxor=0;
        for(int i=0;i<=nums.length;i++){
            rangexor^=i;
        }
        for(int num:nums){
            arrxor^=num;
        }
        return rangexor^arrxor;
    }
}
Time complexity: O(n)
```
