# Approach

## XOR Approach

- Asked to use linear runtime complexity and constant space complexity
- Using XOR we can find the unique element

```java
class Solution {
    public int singleNumber(int[] nums) {
        int out=0;
        for(int i=0;i<nums.length;i++){
            out^=nums[i];
        }

        return out;
    }
}
Time complexity:O(n)
Space complexity:O(1)
```
