# Approach

## Direct Mapping Approach

- If m*n not equal to original array's length, then conversion is not possible, so return empty 2D array.
- Otherwise, Iterate 2D array row-wise and assign elements sequentially from original array. 

```java
class Solution {
    public int[][] construct2DArray(int[] original, int m, int n) {

        int[][] res=new int[m][n];
        if(m*n!=original.length)
        {
            return new int[0][];
        }
        int k=0;
        for(int i=0;i<m;i++){
            for(int j=0;j<n;j++){
                res[i][j]=original[k++];
            }
        }
        return res;
    }
}
Time complexity: O(n)
Space complexity: O(1)
```
