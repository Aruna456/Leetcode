# Approach

## Two pointer Approach

### Swap

- Initalize l and r as 0.
- Iterate r throughout the Array, check if value at r!=0, if is, then swap it with l and increment l.
- else increment r.

```java
class Solution {
    public void moveZeroes(int[] nums) {
        
        int l=0,r=0;

        while(r<nums.length)
        {
            if(nums[r]!=0)
            {
                int temp=nums[r];
                nums[r]=nums[l];
                nums[l]=temp;
                l++;
            }
            r++;
        }
    }
}
Time complexity:O(n)
Space complexity:O(1)
```

### Overwrite

- Instead of swapping, If r finds a non-zero element assign it to l.
- Once r reaches the end means all the non-zero are posistioned to front, fill the Array from i to n with 0.

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int i=0;
        for(int j=0;j<nums.length;j++){
            if(nums[j]!=0){
                nums[i]=nums[j];
                i++;
            }
        }
        Arrays.fill(nums,i,nums.length,0);
    }
}
Time Complexity:O(n)
Space Complexity:O(1)
```
